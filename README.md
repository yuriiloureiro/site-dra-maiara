# Dra. Maiara Website - Clinical Psychology & Neuropsychology

This is the official website for Dra. Maiara, a clinical psychologist specialized in neuropsychology. The site is designed to provide an elegant, accessible, and responsive digital experience that reflects the professional’s visual identity and values.

---

## Technologies Used

- **Next.js 15** (App Router) — modern React framework for SSR and SSG.
- **TypeScript** — static typing for improved safety and productivity.
- **Tailwind CSS** — utility-first CSS framework for fast and responsive styling.
- **Framer Motion** — smooth animations and micro-interactions for enhanced UX.
- **Lucide React** — lightweight and customizable SVG icon library.
- **Next/Image** — optimized and intelligent image loading.

---

## Features

- Responsive, mobile-first layout.
- Main sections: Hero, About, Services, FAQ, Contact, and Footer.
- Animations and transitions for a refined visual experience.
- SEO optimized with metadata and Open Graph configuration.
- Enhanced accessibility with ARIA attributes and keyboard navigation.
- Continuous deployment via GitHub and Vercel for preview and validation.

---

## Getting Started Locally

1. Clone the repository:
   ```bash
   git clone https://github.com/your-username/site-dra-maiara.git
   cd site-dra-maiara
   ```

## Getting Started

First, run the development server:

```bash
npm run dev
# or
yarn dev
# or
pnpm dev
# or
bun dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

Deployment and Validation
The project is configured for automatic deployment on Vercel, connected to the GitHub repository.
The client can access the Vercel-generated link to preview and validate the site before final publication.
After approval, the site will be published on the client’s Hostinger hosting, where the domain and account are already set up.
